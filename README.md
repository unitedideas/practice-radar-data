# Practice Radar data

Versioned public edition receipts and samples for [Practice Radar](https://actablesite.com/practice-radar), plus encrypted per-subscriber weekly editions.

- `public/summary.json`: source and build receipt
- `public/sample.json` and `public/sample.csv`: 15-row public sample
- `paid/`: AES-256-GCM encrypted subscriber artifacts; filenames and contents do not expose Stripe checkout session IDs

Source: CMS NPPES weekly V2. An NPI does not prove licensure, credentialing, active operation, independence, demand, or buying intent.
